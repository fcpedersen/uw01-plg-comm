 This is a simple WordPress CPT commodities plugin. This plugin based on displaying commonly shipped items and three category types they belong to

== Description ==

Custom Post Type plugin for <strong>displaying commonly shipped items</strong> and three category types they belong to

== Installation ==

1. Upload `cpt-commodities.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= How to activate this plugin?

Install this plugin and add custom posts form cpt commodities.
